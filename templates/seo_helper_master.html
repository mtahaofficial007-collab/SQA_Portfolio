Project Overview

Welcome to the SEO Master Dashboard, a modern Flask-based web application designed to help users manage and optimize their SEO performance. This project features a sleek, user-friendly interface with a homepage, feature pages (About, Services, Contact), and a secure dashboard with multiple tools for SEO analysis. User authentication (login/signup) is implemented with data stored in a PostgreSQL database.
